For at deploye projektet køres følgende kommandoer:

 $ cd chatinator3000
 $ npm install 
 $ npm run build
 $ firebase deploy

Tilfå siden på https://makeabletestlasse.web.app/


For at starte projektet lokalt køres følgende kommandoer

 $ cd chatinator3000
 $ npm install 
 $ npm run serve

Tilgå siden på httphttp://localhost:8080/
